Playlist Generator

This project was made as part of the Summer School on Industrial AI at VIVES International School for Applied Sciences. I worked on this project with a relatively small team, and I was given the task to work on some of the python code, particularly aspects related to using the Spotify API and the OpenAI API. It was a novel experience as I've never used those before.

It was also fun trying to debug and study how I could extract data from the Spotify API.

Available as a Github repository for those who are interested to make their own picture->playlist app

Features

  • - Takes in a picture, preferrably of you wearing a t-shirt

  • - Takes into consideration the weather

  • - Generates a spotify playlist for you
